# E-Commerce Cart & Checkout QA Automation (Java)

This project demonstrates QA automation of an e-commerce workflow using Java + Selenium + TestNG.

## Features Tested
- Adding products to cart
- Quantity updates
- Removing items
- Checkout process
- Promo code application
- Edge case handling

## Folder Structure
- **src/test/java/com/qa/automation/** – Java Selenium scripts
- **test_cases/** – Manual test cases & bug reports
- **pom.xml** – Maven dependencies

## How to Run
1. Install JDK 17+
2. Clone the repo
3. Run `mvn clean test` (from project root)
